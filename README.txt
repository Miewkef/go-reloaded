создай папку `texts`, в нем текстовый файл
затем с помощью команды `cd main` на папке main запусти команду `go run . {имя читаемого файла.txt} {имя нужного файла.txt}`

#Доступые функции
{значеие от 0 до 1} (bin) - переводит число из двоичной системы в десятичную
{значеие от 0 до 9 и от a до f} (hex) - переводит число из шестадцатеричной системы в десятичную
